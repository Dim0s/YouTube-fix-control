A browser extension that fixes the following bugs:

1 - extension for Chromium: fixing a bug due to which on youtube after rewinding or changing the volume with the mouse, due to a change in focus, the functions of the cursor keys on the keyboard changed

2 - when rewinding the video with the mouse and quickly moving the cursor along the progress bar, after the video loads, the previews stop displaying.

to install, [download](https://github.com/Dim0s/YouTube-fix-control/raw/refs/heads/main/extension/YouTube%20fix%20control.zip) and unpack it, in the browser in the extensions tab, enable Developer Mode and click on Load unpacked extension, specify the path to the downloaded and unpacked folder, after installation, the extension is automatically enabled only for the YouTube site

расширение для браузера исправляющее следующие баги:

1 - после перемотки или изменения громкости мышкой, из-за смены фокуса, менялись функции клавиш курсора на клавиатуре.

2 - перематывая видео мышкой и резко перемещая курсор по прогресс бару, после прогрузки выдео, перестают отображаться превью.

для установки [скачайте](https://github.com/Dim0s/YouTube-fix-control/raw/refs/heads/main/extension/YouTube%20fix%20control.zip) и распакуйте, в браузере во вкладке расширений включите Режим разработчика и нажмите на Загрузить распакованное расширение, укажите путь к загруженной и распакованно папке, после установки расширение автоматом включается только для сайта ютуба
